## 🤝 기여 가이드

1. 🍴 Fork → 또는 안 해도 됨, 맘대로
2. 🌿 브랜치 생성 → 이름 규칙: `아무거나`, 추천 `fix/고양이가 키보드 밟음`
3. 💻 하고 싶은 것 하기 → 아무것도 안 하는 것도 포함
4. 📤 PR 제출 → 또는 main에 직접 push (샌드박스니까 은행 아님)
5. 🎉 머지 기다리기 → 또는 PR 목록에서 자연스럽게 노화되는 것 기다리기

> ⚠️ **최고 경고**: 망가뜨렸으면 대단한 것 💪, 안 망가뜨렸으면 진지하게 안 한 것 🤠
