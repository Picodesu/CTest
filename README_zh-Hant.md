## 🤝 協作指南

1. 🍴 Fork → 或者不 Fork，隨你
2. 🌿 建立分支 → 命名規則：`任意`，推薦 `fix/我的貓踩鍵盤`
3. 💻 做你想做的事 → 包括什麼都不做
4. 📤 提交 PR → 或者直接推 main（我們是沙盒，不是銀行）
5. 🎉 等合併 → 或者等它在 PR 列表裡自然老去

> ⚠️ **終極警告**：搞壞了算你厲害 💪，搞不壞算你沒認真搞 🤠
