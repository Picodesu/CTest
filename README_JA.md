## 🤝 コントリビューションガイド

1. 🍴 Fork → しくはさくねそれため、お好みほす
2. 🌿 ブランチを作成 → 命名ルール：`あなたろ！`、おすすめ `fix/猪がキーボード踏んだ`
3. 💻 やりたいことをやる → なにもしないのも含まれる

5. 🎉 マージを待つ → しくはPRリストで自然に朽ちるのを待つ
