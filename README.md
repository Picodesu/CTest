## 🤝 协作指南

1. 🍴 Fork → 或者不 Fork，随你
2. 🌿 创建分支 → 命名规则：`任意`，推荐 `fix/我的猫踩键盘`
3. 💻 做你想做的事 → 包括什么都不做
4. 📤 提交 PR → 或者直接推 main（我们是沙盒，不是银行）
5. 🎉 等合并 → 或者等它在 PR 列表里自然老去

> ⚠️ **终极警告**：搞坏了算你厉害 💪，搞不坏算你没认真搞 🫠
